Internal Tools · Documentation

Utility Tool
Guide.

Three quick-action tools for the CS team — Undo Reverse Shipment 2.0 for billing or refunding specific SKUs, Undo Self Checkout for reverting SCO orders, and Reset Return for reverting or reopening a return. Scroll to see each one, step by step.

Scroll
Tool 01 · RS Channel

Undo Reverse
Shipment 2.0.

Bill or refund specific SKUs on an RS order before FC finalizes the return. Five steps.

Before you begin

How to use this.

  1. Only orders that went through RS and are still in a billed state can be processed.
  2. Use this only for orders that you want to quickly refund or bill the customer.
  3. This is only a pseudo state. When FC processes the return, it will bill accordingly:
    • Un-sold refund the customer for that specific SKU
    • Un-reverse shipped bill the customer for that specific SKU
    • Un-reverse damaged bill the customer for that specific SKU
01
Step One

Paste the order number.

Drop the RS order # into the input and hit Submit. The tool validates that the order went through RS and is still in a billed state before loading anything.

Tool 03 · Returns

Reset Return.

Revert the last return or reopen an order for additional return items. Two scenarios — one checkbox decides which.

Before you begin

How to use this.

  1. Order should be in returned state.
    • Checkbox Unchecked — Reverts the last return, allowing the previously returned item to be processed again.
    • Checkbox Checked — Reopens the order, allowing an additional item to be added for return.
  2. Note: Only orders within the 7 day window can still be processed.

Scenario A

When to uncheck.

Use Unchecked when the previously returned item needs another pass — maybe it was mistaken or needs restocking review. All items stay on the order and revert to processable state.

Scenario B

When to check.

Use Checked when a customer wants to return an additional item beyond what was originally authorized. The order moves to returning and extra items can be added.

That's all three tools.

Undo Reverse Shipment 2.0 for RS orders, Undo Self Checkout for SCO orders, and Reset Return for revert-or-reopen on returns. Remember the guardrails — RS needs billed state, SCO needs self_checkout_processed, and Reset Return only works within the 7-day window.
